# Lapse - Life Progress Menu Bar App for macOS

# Setup

#### Install Bitbar
    $ brew cask install bitbar

#### Adding Plugins
- Open Bitbar and select Change Plugin Directory
- Choose the directory where lapse.30m.sh is saved
